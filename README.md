# Admin Dashboard (Bootstrap)

Admin dashboard UI built with plain HTML and Bootstrap 5. Includes a main overview page and a dedicated analytics page with SVG-based charts.

## Pages
- `home.html` — Overview dashboard
- `analytics.html` — Analytics view with charts

## Styles
- `css/dashboard.css` — Custom theme and layout styles

## Run Locally
Open `home.html` or `analytics.html` in a browser.

## Notes
- Uses Bootstrap via CDN.
- No build tools or JavaScript frameworks required.
